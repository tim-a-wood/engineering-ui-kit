# Aircraft Performance sample

This built-in sample exercises the Capabilities workflow with a generic,
no-brand long-range business aircraft. Its completed product-interview draft
lives in `capabilities/application-specification.json` and is seeded into the
desktop app on first launch without being pre-approved.

The small static target UI exists so later Connect and Verify stages have a
real local application to inspect. From this directory:

```sh
npm install
npm run dev
```

The sample is deliberately illustrative rather than certified performance
software or operational guidance.
