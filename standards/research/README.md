# Research

## Purpose

This folder records how public-source research informs Phase 2 machine-readable contracts.

The research folder does not contain copied third-party design-system text, icons, images, code, or branded visual identity. It records reference posture only.

## Files

| File | Purpose |
|---|---|
| `component-reference-map.md` | Maps manifest component families to reference-backed, PRD-backed, or inferred engineering inclusion rationale. |

## Interpretation rule

A public reference can support inclusion, accessibility expectations, documentation structure, or implementation primitive evaluation. It does not make the Engineering UI Kit a fork, clone, or branded derivative of that reference.
